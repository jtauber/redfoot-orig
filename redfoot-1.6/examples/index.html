Redfoot > Examples

Redfoot Examples

To Run

From the examples directory: 
python ../redfoot.py <init_script>

where

init_script is the .init file for one of the following examples.

Examples

Hello World
A simple example of how to extend a node with an HTTP interface.
Hello World Redcode
A simple example of how to extend a node with an HTTP interface using Redcode.
Generic
A web based generic RDF editor
RSS
An example of an RSS HTTP app and module for working with RSS 1.0 RDF.
Sniffer
The link sniffer is (currently) a standalone rednode that will parse web pages for links adding any that it finds to itself. The node sniffs pages as indicated via triples added to it.
Grid
An example for showing an example GridInstance Redfoot Module
Static
An example that uses the redmodule.static.Static Redfoot module to serve up files from the filesystem.
