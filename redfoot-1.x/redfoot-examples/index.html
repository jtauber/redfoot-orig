Rednodes

Examples of complete runnable nodes. The nodes may either be stand alone nodes or nodes that will form a p2p network.
Link Sniffer
The link sniffer is (currently) a standalone rednode that will parse web pages for links adding any that it finds to itself. The node sniffs pages as indicated via triples added to it. This node can be run via the link sniffer init script.
