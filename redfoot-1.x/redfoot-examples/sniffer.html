Link Sniffer Readme

To Run

(from this directory) 
python ../../redfoot.py init_script
or (from redfoot-1.5.1) 
python redfoot.py redfoot-examples/link_sniffer/init_script

FILES

init_script
An script that initializes Redfoot so that it is running the link sniffer.
__init__.py
This package's __init__, as link_sniffer is a top level python package.
node.py
A node that knows how to parse html pages via a parser (see below) which adds triples to the node for new links that it finds.
parser.py
A parser which knows how to parse HTML looking for links.
http_app.py
An http interface for displaying the link data that is accumulated in the node.
links.rdf
The data file to which the RDF is persisted.
