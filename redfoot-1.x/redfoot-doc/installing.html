Installing Redfoot

Prerequisite

Python version 2.2.x is required and needs to be installed first.

Running Install

From the root directory run the following command:

python install.py

Since this is a development version the redfootlib package is not installed. This is due to the fact that the distutil based setup.py which is used does not handle uninstalling packages and also should simplify debugging.

The above command should install the required version of the Medusa package if you do not yet have a version installed. If you already have a version of Medusa installed you will likely have to manualy update it to the version included in this release, namely, 0.5.2-rf1. Which at the time of this release is the latest Updated Medusa Release plus one bug fix Redfoot Required. The bug fix has been made to the Medusa CVS and so should be in 0.5.3 when it comes out.
