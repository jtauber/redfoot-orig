Running Redfoot

Run

python <relative_path>/redfoot.py <init_script>

Where

relative_path
the relative path to redfoot.py
init_script
a relative of absolute pathname of a Redfoot init script file

Example

From the top level directory: 

python redfoot.py redfoot-example/hello_world.init

You should see something along the lines of: 

Redfoot Command Line
warning: Computing default hostname
info: Medusa (V1.5) started at Tue Apr 23 03:26:12 2002
        Hostname: eikco-1
        Port:9090

Started server at: http://localhost:9090/
RF>

You should now be able to see the Hello World app running by going to http://localhost:9090/ or type in help at the RF> prompt for a list of commands.

See Redfoot Examples or Redfoot Examples (Online Version) for the list of init scripts.
