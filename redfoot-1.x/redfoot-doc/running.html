Running Redfoot

Run

python <relative_path>/redfoot.py <init_script>

Where

relative_path
the relative path to redfoot.py
init_script
a relative of absolute pathname of a Redfoot init script file

Example

From the top level directory: 

python redfoot.py redfoot-example/hello_world.init

See Redfoot Examples or Redfoot Examples (Online Version) for the list of init scripts.
