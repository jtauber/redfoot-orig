Redfoot

Redfoot Documentation

Redfoot is an extensible RDF server for building {peer|distributed} networks of data sharing nodes upon which applications may be built.

At the highest level Redfoot consists of two pieces, namely, an {RDF Container|RDF Application Server} for running the nodes that form the network and a library from which nodes may be extended with specialized application specific functionality. The Container is simply called Redfoot and the library redfootlib. See the architecture section for a more in depth description.

NOTE: The following documentation is available at http://redfoot.sourceforge.net/doc/. Please check back regularly for updates.
