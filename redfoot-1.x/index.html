Redfoot

Redfoot 1.5.1 Readme

Redfoot is an extensible RDF server written in python for building networks of distributed data sharing nodes and upon which applications may be built. It is being developed by Daniel Krech and James Tauber.

DOCUMENTATION

Please see either the Local Redfoot Documentation or the Online Redfoot Documentation for getting started and other documentation.

WHERE TO GET THE LATEST REDFOOT

See http://redfoot.sourceforge.net/
