Known Issues

These issues are known to pertain to 0.9.9 and 0.9.99-pre1.

Can't access via 'localhost' on MacOS X

PROBLEM: If Redfoot is run with the hostname of 'localhost', it cannot be accessed by a browser using 'localhost' as the hostname.

SOLUTION: Use 127.0.0.1 to access Redfoot with your browser.

"ImportError: No module named thread" running on Cygwin

PROBLEM: Some versions of Cygwin (1.3.2 for example) come with a version of Python compiled without threads.

SOLUTION: Uninstall the cygwin python package and get a windows version from python.org that is compiled with thread support.
